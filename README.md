## Основные возможности

### Каталог и Товары
*   Просмотр списка товаров с пагинацией и удобной фильтрацией.
*   Детальная страница товара.

### Корзина
*   **Гибридное хранение**:
    *   Для гостей: товары хранятся в сессии.
    *   Для авторизованных пользователей: товары сохраняются в базе данных.
*   **Автоматическая синхронизация**: При входе в аккаунт товары из "гостевой" корзины объединяются с сохраненной корзиной пользователя.
*   AJAX-обновление количества товаров и итоговой суммы.

### Избранное
*   Добавление и удаление товаров из избранного.
*   **Автоматическая синхронизация**: Избранное также синхронизируется при входе в систему.
*   Динамический счетчик избранных товаров в шапке сайта.

### Заказы
*   **Оформление заказа**: Создание заказа на основе выбранных товаров из корзины.
*   **История заказов**: Просмотр списка всех заказов пользователя.
*   **Детализация**: Хранение информации о доставке (адрес, город) и статусе оплаты.

### Категории
*   Иерархическая структура категорий.
*   Удобная навигация и фильтрация товаров по категориям.

### Пользователи
*   Регистрация и авторизация.
*   Вход через социальные сети.
*   Личный кабинет пользователя.

## Технологии
*   **Backend**: Python 3.12, Django 5.1
*   **Database**: PostgreSQL
*   **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla)

## Структура проекта
*   `myapp` - Основное приложение (главная страница, отображение товаров).
*   `users` - Управление пользователями (аутентификация, профили).
*   `cart` - Логика корзины (добавление, удаление, API).
*   `favorites` - Приложение избранного.
*   `orders` - Управление заказами.
*   `category` - Модели и логика категорий товаров.

## Установка и запуск

### 1. Создание виртуального окружения
#### Windows:
````bash
python -m venv venv
venv\Scripts\activate
````
#### Linux/macOS:
````bash
python -m venv venv
source venv/bin/activate
````

### 2. Установить зависимости
````bash
pip install -r requirements.txt
````

### 3. Применить миграции
````bash
python manage.py migrate
````

### 4. Запустить сервер
```bash
# Стандартный запуск (HTTP)
python manage.py runserver

# Запуск с HTTPS (требуются сгенерированные сертификаты)
python manage.py runserver_plus --cert-file cert.crt
```

### 5. Открыть в браузере
* HTTP: http://127.0.0.1:8000/
* HTTPS: https://siterandomshop.ru:8000/

## Настройка HTTPS с доменом siterandomshop.ru

### 1. Открыть файл hosts
#### Windows:
```
C:\Windows\System32\drivers\etc\hosts
```
#### Linux/macOS:
```
/etc/hosts
```

### 2. Добавить строку
```
127.0.0.1       siterandomshop.ru
```

### 3. Сгенерировать SSL сертификаты
```bash
openssl req -new -newkey rsa:2048 -days 365 -nodes -x509 -keyout cert.key -out cert.crt
```

### 4. Запустить с HTTPS
```bash
python manage.py runserver_plus --cert-file cert.crt
```
